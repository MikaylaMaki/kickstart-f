# Kick Start Notes

Test input is passed via stdin

Use main.rs as a simple scaffold for reading and writing values from stdin and stdout

probably better to use `rustc file.rs` to pipe to stdin:

`pushd src; rustc main.rs; ./main < sample.txt; popd;`
